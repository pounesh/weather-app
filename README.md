 🌦️ Weather App

A simple weather application built with HTML, CSS, and JavaScript.

 ✨ Features
- Search weather by city name
- Displays:
  - Temperature
  - Humidity
  - Wind Speed
  - Weather Icon
- Responsive design
- Shows weather condition (Cloudy, Clear, Rain, etc.)
-  Real-time weather data using OpenWeatherMap API
- Error message for invalid city names



 🛠️ Technologies
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API
